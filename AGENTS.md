# UniSchedule AI — Agent Instructions

This repository is a **Final Year Project**: an intelligent university meeting scheduler with a research-led dissertation.

## Start here

1. Read [`docs/PROJECT-NOTES.md`](docs/PROJECT-NOTES.md) for scope, RQs, MoSCoW, and timeline.
2. Cursor rules in [`.cursor/rules/`](.cursor/rules/) apply automatically (`unischedule-core` always on).

## Stack

- **Frontend:** Next.js + TypeScript  
- **Backend:** Laravel REST API  
- **Database:** MySQL  
- **Auth:** Laravel Sanctum  

## Planned structure

```
frontend/   # Next.js app
backend/    # Laravel API
docs/       # FYP documentation
```

## Research focus

Implementation must support **evaluation** (manual vs AI scheduling, clash metrics, UAT). See `docs/supervisor-submission/02-research-foundation.md` and `06-evaluation-security-ethics.md`.
